# 移动Web

## 移动端布局方案

- 伸缩布局 flex布局 在新框架中 bootstrap 4.0  携程
- 百分比布局 %  比较适合电商网站  京东移动端
- rem布局+sass  rem布局是现在用的最多的移动端布局方式 可以适配各个不同的手机 psd切图
- 响应式布局 特别特别简单的网站才能用响应式 一套html页面 多个终端的css样式 在哪个终端 应用哪个终端的样式

## 京东移动端

### 项目技术点

- 布局方案: 百分比布局

- normalize.css 是最最流行的浏览器重置样式
