#### box-sizeing 盒子尺寸

- 作用: 当给盒子重新添加padding border的时候 重新计算盒子的宽和高

- 应用场景: * {box-sizing: border-box}

- 属性:box-sizing
 - content-box(默认值)
 - border-box(只要用box-sizing)
