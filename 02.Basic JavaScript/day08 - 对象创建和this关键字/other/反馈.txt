  预解析的过程
    1.js的执行 一共分两个过程: 一个是预解析 一个是执行
    2.变量的预解析规则是: 把变量的声明提升到它自己作用域的最前面 赋值不提升
    3.函数的预解析规则是: 把函数的声明提升到它自己作用域的最前面  调用不提升
    4.如果函数和变量同名 函数优先
    5.函数的预解释中如果函数是 var  变量名 = function () {} 先提升变量名 在提升 function
